利用位运算来找不同。

一个数两次异或同一个数就是其本身，所以用一个0分别和 `s` 和 `t` 中每一位异或，得到的结果就是新增加的值。

```Go
func findTheDifference(s string, t string) byte {
    var diff byte
    for i := 0; i < len(s); i++ {
        diff ^= s[i]
        diff ^= t[i]
    }

    return diff ^ t[len(t) - 1]
}
```
