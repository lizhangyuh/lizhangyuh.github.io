动态规划

从后往前考虑。

时间复杂度：O(N^2)
空间复杂度：O(1)

```Go
func jump(nums []int) int {
    count := 0
    r := len(nums) - 1

    for r > 0 {
        for i := 0; i < r; i++ {
            if nums[i] >= r - i {
                r = i
                count++
                break
            }
        }
    }

    return count
}
```

空间复杂度：O(1)
